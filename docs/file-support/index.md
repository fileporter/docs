# Text

All Text files are supported and code-files should get code-highlight.

# Images

All common Image-formats should be supported.

# Videos

All common Video-formats should be supported.

<small>There is an option for Videos with no sound and a duration of less than ~60s to be handled like .gif files.</small>

# Special File Formats

## .md

Markdown documents are rendered.

## .url

is supported

<sub>File structure:</sub>
```url
[InternetShortcut]
URL=https://fileporter.github.io/docs/
```

## .pdf

is supported
